## 📌 关于 Paper Digest

**Paper Digest** 是一个全自动的 ArXiv 论文追踪与总结系统，专注于 **Agent Memory** 领域的前沿研究。

### 🎯 覆盖主题

- **Agent Memory Architecture** - 智能体记忆架构设计
- **Long-term Memory Systems** - 长期记忆存储与检索
- **Memory-augmented LLMs** - 记忆增强的大语言模型
- **Multi-agent Memory Coordination** - 多智能体记忆协同
- **Memory Safety & Privacy** - 记忆安全与隐私保护
- **Continual Learning for Agents** - 智能体持续学习

### ⚡ 工作流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ ArXiv API │───→│ 每日查询  │───→│ 主题筛选  │───→│ AI生成摘要│───→│ 自动发布  │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
```

### 📅 更新频率

- **每日自动更新** - 北京时间每天凌晨扫描 ArXiv 当日新论文
- **周末特别版** - 每周五发布当周研究热点综述
- **月度总结** - 每月回顾重要进展和技术趋势
